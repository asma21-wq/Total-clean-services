# Prestige Clean Services Website

Site web statique pour Prestige Clean Services, entreprise de nettoyage à Strasbourg.

## Structure du site

- `index.html` : Page d'accueil
- `nettoyage-professionnel.html` : Page nettoyage professionnel
- `nettoyage-automobile.html` : Page nettoyage automobile
- `contact.html` : Page contact
- `css/style.css` : Feuille de style
- `js/script.js` : Scripts JavaScript
- `images/` : Dossier pour les images
- `sitemap.xml` : Sitemap pour SEO
- `robots.txt` : Fichier robots pour SEO

## Instructions d'installation

1. Téléchargez tous les fichiers dans un dossier.
2. Ajoutez les images nécessaires dans le dossier `images/` :
   - `favicon.ico` : Icône du site
   - `hero-bg.jpg` : Image de fond pour la section hero (optionnel)
   - `whatsapp-icon.png` : Icône WhatsApp (téléchargeable gratuitement)
3. Remplacez `GA_MEASUREMENT_ID` par votre ID Google Analytics réel.
4. Hébergez le site sur un serveur web (ex: GitHub Pages, Netlify, etc.).

## SEO et optimisation

- Le site est optimisé pour le mobile.
- Balises meta title et description présentes.
- Sitemap et robots.txt inclus.
- URLs propres.
- Intégration Google Analytics.
